# DSA-IN-JAVA
Java implementation of linear search to find the index of a given element in an array

OVERVIEW 
This Java program takes an array as input and finds the index of a given element (x) using Linear Search.

🚀 Example

### Input
Array = [1, 2, 3, 4, 5]  
x = 5  

### Output
Element found at index 4

(Note: Indexing starts from 0 in Java.)

🧠 Algorithm Used
Linear Search

⏱ Time Complexity
- Best Case: O(1)
- Worst Case: O(n)

📦 Space Complexity
O(1)

🛠 Technologies Used
- Java
